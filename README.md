# Melhores Praias do Brasil
Este projeto é um carrossel aumático que mostra o rank das 5 melhores praias, de acordo com uma pesquisa, desenvolvido com o objetivo de praticar a utilização de HTML, CSS e JavaScript.

Funcionalidades:
• Carrossel automático
• Loop infinito
• Design simples
• Responsividade

Link do projeto: https://melhores-praias.vercel.app/
